Individual section(s)

Guideline: all layers documenting a component should use "Section" as their layer name. They should have 64px outer spacing and a 32px gap.

Top level frames

Guideline: all top level frames should have 0px spacing and 0px gap, since the spacing is contained in the sections.

Guideline: Add stop emoji prefix to frame names for marking purposes.
